Orbit Guides was born from a real problem: as frequent travelers and Airbnb guests, Julia Bacchi and I noticed most hosts either sent too many messages or provided too little information. We built the tool we wished we'd found.

Built from scratch as a full-stack Next.js application, Orbit Guides features:

- **Admin Dashboard** — manage all properties from a single panel with property info, rules, local tips, and amenities
- **QR Code Generation** — each guide gets a unique QR code that hosts can print or display at the property
- **Auto-translation** — guides automatically translate into the guest's language, supporting guests from over 50 countries
- **Real-time Updates** — change Wi-Fi passwords or any info and it updates instantly, no reprints needed
- **Analytics** — hosts can see what guests access most and improve their local tips
- **Subscription Plans** — Freemium, Starter (R$49/mo), Essential (R$149/mo), and Pro (R$279/mo) tiers

The platform serves hosts across 5+ countries and integrates seamlessly with Airbnb, VRBO, and other vacation rental platforms. Built with Next.js, TypeScript, Tailwind CSS, and a focus on performance, scalability, and delightful user experience.
